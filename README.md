# NYC-Yellow-Taxi-Project
A data analysis project analyzing tipping behavior in NYC 2022 and building a linear regression model to predict tip amounts based on certain indicators (payment type, pickup zone, fare amount, trip duration).

Download January 2022 NYC Yellow Taxi Trip Records parquet file from https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.pageLinks
